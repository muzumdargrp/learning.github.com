# Learning Semantics Static Website

This is a GitHub Pages-ready static website for Learning Semantics.

## How to deploy on GitHub Pages

1. Create a new GitHub repository, for example:
   `learning-semantics-site`

2. Upload these files to the repository root:
   - `index.html`
   - `styles.css`

3. In GitHub:
   - Go to **Settings**
   - Go to **Pages**
   - Under **Build and deployment**, choose:
     - Source: `Deploy from a branch`
     - Branch: `main`
     - Folder: `/root`

4. Save.

Your site will be available at:
`https://YOUR-USERNAME.github.io/learning-semantics-site/`

## Custom domain

To use `learningsemantics.com`, add a `CNAME` file containing:

learningsemantics.com

Then configure DNS with your domain registrar.
